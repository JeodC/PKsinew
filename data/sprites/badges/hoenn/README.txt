# Hoenn Gym Badges — Ultimate Pack by RamiroMaldini

This folder contains the **Hoenn region gym badges** from the “Ultimate Pokémon Badges Pack HD,” a fan-made badge icon pack created by **RamiroMaldini**.  

The badges included here are:

- Stone  
- Knuckle  
- Dynamo  
- Heat  
- Balance  
- Feather  
- Mind  
- Rain  

## Source & Attribution

Original artwork: [Ultimate Pokémon Badges Pack HD on DeviantArt](https://www.deviantart.com/ramiromaldini/art/Ultimate-Pokemon-Badges-Pack-HD-336140451)

These badges are included **with attribution** as part of this project.

## Usage

- Filenames have been simplified to just the badge name for easier referencing in code:
  - `stone.png`, `knuckle.png`, `dynamo.png`, `heat.png`, `balance.png`, `feather.png`, `mind.png`, `rain.png`  
- Intended for use in this project as UI icons for Ruby, Sapphire, and Emerald.
